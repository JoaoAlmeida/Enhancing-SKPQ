The following describes the documents and folders related to the article "Personalizing the Top-k Spatial Keyword Preference Query with Textual Classifiers" (https://doi.org/10.1016/j.eswa.2020.113841).

Folders 'hotelsDubai' and 'hotelsLondon' contain the set of reviews describing each hotel in Dubai and London.

The files 'clean', 'location', 'room', 'service', and 'value' are user profiles that simulate a user with a specific interest as described in the article. All these profiles were created by using reviews from hotels in Dubai.
'serviceLondon' was created to simulate a user interested in hotels with good service, using reviews of hotels in London.
